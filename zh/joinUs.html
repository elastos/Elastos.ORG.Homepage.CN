高级区块链开发工程师

岗位职责：

1. 区块链共识及其他核心机制方案调研、讨论、开发、维护；

2. git hub相关文档编写维护；

3. 社区技术培训和交流；

岗位要求：

1. 计算机及相关专业本科以上学历；

2. 3年以上软件开发经验；

3. 熟悉linux编程环境，精通go或c++语言；

4. 有socket网络编程经验；

5. 有多线程编程经验；

6. 熟悉各种数据结构及算法；

7. 良好的编程习惯，对内存、多线程、网络等方面的bug和不健壮性有深切体会和处理经验；

8. 有软件架构思维；

9. 对密码学，安全协议和加密算法有原理上的理解以及相关应用开发经验；

10. 深入了解至少一个区块链产品（bitcoin/ethereum/eos/fabric）的工作原理；

11. 有区块链产品开发经验优先考虑；

12.有p2p通讯相关开发经验优先考虑；

区块链高级测试工程师

岗位职责：

1. 负责需求分析，测试规范的制定与计划，分配并指导测试人员开展测试活动；

2. 组织实施产品测试过程，负责测试工作的顺利开展；

3. 解决测试过程中的重大问题；

4. 配合项目负责人组织产品交付，对最终质量负责；

岗位要求：

1. 具备至少3年以上测试团队管理经验；

2. 5年以上 互联网、通信、IT相关领域项目测试计经验；

技术要求：

1. 计算机、电子、数学等相关专业本科及以上学历；

2. 精通Java、C++等面向对象语言；

3. 扎实的软件开发和测试技术，精通具备自动化测试和性能测试专业知识和经验；

4. 熟悉Linux基本知识，了解虚拟化等技术尤佳；

C/C++高级工程师

岗位职责：

负责亦来云的系统层平台服务和应用的开发

1. 根据平台需求，进行平台服务、SDK和应用的开发；

2. 根据产品功能模块设计，编码实现各模块功能，并确保开发质量；

3. 参与接口规范制订、技术文档编写；

岗位要求：

1. 本科及以上学历，计算机或相关专业；关注区块链及去中心化技术，对区块链、去中心化应用充满激情；

2. 3年以上C/C++开发经验；熟悉内存管理，具备并发和网络开发能力和经验；

3. 具有跨平台开发能力和经验；

4. 熟悉密码学和非对称加密体系；

5. 熟悉开源库，并有开源库使用经验；

6. 能熟练使用调试、性能分析工具来调试和优化代码；

7. 具有移动终端平台开发经验者优先；

3. 参与接口规范制订、技术文档编写；

岗位要求：

1. 本科及以上学历，计算机或相关专业；关注区块链及去中心化技术，对区块链、去中心化应用充满激情；

2. 3年以上Java开发经验，2年以上Android开发经验；有丰富的Android应用架构经验，可以独立承担架构设计，并主导项目；
